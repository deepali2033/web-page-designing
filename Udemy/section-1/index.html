What is HTML?

HTML stands for HypreText Markup Language, it's markup language that web developers use the stracture and discribe the content on web pages(not a programming Language)

HTML consists of elements that discribe different type of content: paragraph, heading,links images,videos etc.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
